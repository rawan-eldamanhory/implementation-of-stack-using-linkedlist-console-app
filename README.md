# Stack Using Linked List (C#)

This project demonstrates a manual implementation of the **Stack data structure** using a **Singly Linked List** in C#.

## Features
- Push elements onto the stack
- Pop elements from the stack
- Peek the top element
- Check if the stack is empty
- No use of built-in Stack class

## Concepts Used
- Linked List
- Stack (LIFO principle)
- Object-Oriented Programming
- Exception Handling

## Example Output
Popped: 50
Popped: 40
Top element: 30

## Technology
- C#
- Console Application
- Visual Studio

## Purpose
This project is created for learning and practicing **data structures** using C# without built-in collections.

## How to Run
1. Open the project in Visual Studio
2. Build and run the console application
